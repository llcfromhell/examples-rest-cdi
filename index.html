Java - Rest + CDI

Recently, a colleague of mine needed to implement some RESTful Web Services on the JavaEE platform. At some point he asked for my help making the code to run.

After some dull tries I went home and finished a sample example so I could show and help him finish his task.

Tech stuff

I used the following:

  • Eclipse Mars 2 4.5.2
  • Wildfly 10.0 Final (JEE 7, Weld impl. for CDI)
  • Jersey Servelet 1.19.1
  • Apache Maven 3.3.9
  • Java JDK 1.8.0_74 64 Bits
  • Windows 10 Home SL
